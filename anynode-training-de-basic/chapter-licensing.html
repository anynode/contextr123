Licensing

Lektion 6 / 22 last update: 14.5.2026

Introduction

If you would like to test anynode first without commitment, the free anynode demo license is automatically available with every anynode installation. The demo license is limited to four sessions and periodically plays a voice announcement. In addition, the functions Media Passthrough and Internal Path Replacement are not available.

Our licensing team can also provide you with more sessions, and you can receive a free LOD key for a personal, time-limited trial license without voice announcements. An LOD key is an activation code that you receive by email after purchase, which allows you to generate your final license. Using an LOD key gives you the advantage of generating the license file independently of time and location.

The demo license can be quickly converted into a full version simply by replacing the license. Therefore, no new installation or configuration is required. The installed demo license is also automatically removed when a new license is installed. Every regularly purchased license always includes one year of Software Upgrade Service, which can be extended as needed.

The Software Upgrade Service allows you to update anynode to the latest released version during this period. There are no restrictions—the determining factor is the date on which the license was activated. From that moment on, the validity period begins. anynode versions released after this period expires cannot be used or can only be used in demo mode.

ℹ
For successfully completing the exercises, it is important that you perform the licensing using the LOD keys provided for your location in your course materials under the “Licensing” section. Without proper licensing, not all exercises will function.

Step-by-step instructions


Step 1 – Request a license using the anynode license assistant

Video step 1 YouTube ID:

The second option, Request a license from the License on Demand Server, is the easiest and fastest way to license your anynode. For this, your anynode machine must be connected to the internet. The licensing assistant will check whether the LOD license server is reachable.

Even if your anynode machine does not have an internet connection, you can still perform the licensing process. In that case, the first option, Import license file, is used. In this training, we will demonstrate the second option.

  • Navigate to Licenses in the left menu.
  • Under Licenses installed, click Add.
  • Select Request a license from the License On Demand server.
  • Click Next.

Step 1.1 – Link the license to the specific hardware ID

Each license must be linked either to the MAC address of the network adapter of your hardware or to the specific hardware ID of the system. In the License Assistant, these values are generally referred to as the list of System IDs.

  • Then select Microsoft Hyper-V Network Adapter as the System ID.
  • Click Next.

Step 1.2 – Enter the LOD key

In the next step, copy the LOD key into the designated field.

  • Click Next.

Step 1.3 – Identification

Your email address is required so that a copy of the generated license can be sent to you. This ensures that you receive all important information for archiving purposes. The Company name field should always contain the end customer’s name.

  • Enter your email address and company name.
  • Click Next.

Step 1.4 – Activate the license

  • In the final step, activate the LOD key online by clicking Activate Now.
  • Click Finish.

Afterwards, you will see an overview of your installed licenses. The license indicator in the top right corner should then display a green status.

Completion of the exercise

In this chapter, you successfully completed the licensing of anynode. You requested a license using the anynode License Assistant, bound it uniquely to the system’s hardware ID, and entered the provided LOD key. By completing the final activation in the frontend, the demo version was converted into a fully licensed anynode installation.

The system is now properly licensed and fully operational—an important prerequisite for productive operation as well as for all further configuration and extension steps during the training.
